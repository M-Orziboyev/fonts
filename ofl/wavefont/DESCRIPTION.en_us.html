Wavefont is a variable font with a weight and round axis for rendering data: waveforms, spectrums, diagrams, bars etc.

Wavefont bars correspond to values from 0 to 100, assigned to different characters:
- 0-9 chars are for simplified manual input with step 10.
- a-zA-Z for extended manual input with step 2, softened at edges a and Z.
- Unicode range U+0100-016F for 0..100 bottom aligned values (convert as char = String.fromCharCode(0x100 + value)).
- Unicode range U+0400-046F for 0..100 center aligned values (convert as char = String.fromCharCode(0x400 + value)).

letter-spacing CSS property with ch units is useful to adjust distance between bars, 1ch === 1 bar width.

To contribute, see github.com/dy/wavefont.
